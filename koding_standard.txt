
1. За увлачење се користе размаци а не табови.
2. Сваки нови ред исте наредбе увлачи се за по један таб. Изузетак су витичасте заграде.
	Нпр.:
		if (i < 0
			|| j > 0)
		{
			printf("%d", i);
			j++;
		}

3. Отворена и одговарајућа затворена витичаста заграда морају бити у истој колони. Блок увек почиње у новом реду.
	Нпр.:
		for (i = 0; i < n; i++)
		{
			x += niz[i]; // тело је увучено, али саме витичасте заграде су у истој колони као и for наредба
		}

4. Декларација једне променљиве у једном реду.
  Погрешно:
   char src, *dst, temp;
  Исправно:
   char src;
   char* dst;
   char* temp;
   
5. Звездица код декларације показивача, односто амперсенд (&) код декларације референци, треба да буду спојени са типом на који се показују..
	Нпр.:
		void* x; а не void *x;
		int& i; а не int &i;
	
6. Бинарни оператори имају размак и са леве и са десне стране. 
	Нпр.:
	 c = a + b;
7. Унарни оператори су прилепљени уз операнд.
	Нпр.:
	 b++;
8. Отворена заграда код позива функција је прилепљена уз назив функције.
	Нпр.:
	 foo(a);
9. Параметри и аргументи функција се одвајају једним зарезом и размаком.
	Нпр.:
		void Foo(int i, float f);  // i, f - параметри функције
		Foo(34, 3.4)               // 34, 3.4 - аргументи функције		

10. Отворена заграда код резервисаних речи (if, while, for, switch) је одвојена од разезрвисане речи.
	Нпр.:
		for (i = 0; i < n; i++)

11. Иницијализовати променљиве у моменту декларације (уколико је то могуће).
